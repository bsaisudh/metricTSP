Prerequisits:

Python 3
Matplotlib
Numpy
Tsplib95 - pip install tsplib95

Contents:

1) 'mtericTSP' Folder - Code for generating tour
2) 'mtericTSP/Data' Folder - *.tsp and *.out.tour files (Input and output data files)
3) 'mtericTSP/Results' Folder - Report and images

Code Execution :

1) Unzip the contents to folder
2) Open metricTSP folder

To find tour for given test files:

4) Execute 'TestFileEvaluvation.py'
	The tour file will be generated in '/metricTSP/Data' folder

To execute the Random evaluvation,

3) Execute 'RandomEvaluvation.py'
	The tour files will be generated in '/metricTSP/Data/Random' folder
